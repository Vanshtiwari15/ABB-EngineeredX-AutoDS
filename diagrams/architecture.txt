                         🎯 RECOMMENDATION ENGINE LOGIC

    Input: Dataset + Task Description
       │
       ├─ Parse Description
       │  └─ Extract keywords for each task type
       │     • classification: "classify", "category", "binary"
       │     • regression: "predict value", "continuous"
       │     • clustering: "cluster", "group", "segment"
       │     • nlp: "text", "sentiment", "language"
       │     • time_series: "forecast", "trend", "temporal"
       │
       ├─ Analyze Data Characteristics
       │  ├─ Target variable analysis
       │  │  ├─ Type (numeric, categorical)
       │  │  └─ Distribution (balanced, imbalanced)
       │  ├─ Feature analysis
       │  │  ├─ Types (numeric, categorical, datetime)
       │  │  └─ Quality (missing values, outliers)
       │  └─ Dataset size assessment
       │     ├─ Small (< 1K): Use simple models
       │     ├─ Medium (1K-100K): Use ensemble
       │     └─ Large (> 100K): Use scalable models
       │
       ├─ Calculate Confidence Scores
       │  ├─ Description score (40% weight)
       │  ├─ Data characteristics (60% weight)
       │  └─ Normalize to 0-100% for each task
       │
       └─ Generate Recommendations
          ├─ Select top-3 models for task type
          ├─ Suggest preprocessing pipeline
          ├─ Recommend evaluation metrics
          └─ Add contextual notes
