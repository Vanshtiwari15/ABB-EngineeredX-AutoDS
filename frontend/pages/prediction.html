JSON Format Help

Required structure for batch predictions

The inference engine parses a JSON array where each object contains column names as keys mapped to row feature values: 

[
  { "age": 28, "income": 55000 },
  { "age": 42, "income": 72000 }
]

Note: Feature headers must match the columns of your trained dataset exactly (case-sensitive).
